# 🔁 Palindrome Checker App

## 📌 Overview

The **Palindrome Checker App** is a console-based Java application that validates whether a given string is a palindrome under different conditions.

This project is designed using an **App-Based Learning approach** to demonstrate how **Core Java concepts and Data Structures** can be applied to solve logical and real-world validation problems.

---

## 🎯 Objective

The objective of the **Palindrome Checker App** is to design and implement a console-based Java application that determines whether a given string is a palindrome under various conditions.

This project helps in:

* Strengthening core programming fundamentals
* Applying data structures for efficient string processing
* Handling multiple input constraints and edge cases
* Writing clean and maintainable code

---

## 🧠 Learning Approach

The application is developed incrementally, where each use case introduces a new concept while reinforcing earlier ones.

Each stage focuses on:

* ✔ Choosing the appropriate approach (String, Stack, etc.)
* ✔ Understanding the problem in different scenarios
* ✔ Improving logic for better efficiency and flexibility

---

## ⚙️ Features

### 🔤 Basic Palindrome Check

* Checks if a string reads the same forward and backward
* Case-sensitive comparison

### 🔍 Advanced Validation

* Case-insensitive palindrome checking
* Ignores spaces and special characters
* Handles numeric and alphanumeric inputs

### 🧩 Multiple Approaches

* Using String reversal
* Using Stack (LIFO principle)
* Two-pointer technique for optimized performance

### ⚠️ Validation & Edge Cases

* Handles empty strings
* Handles single-character inputs
* Exception handling for invalid inputs
